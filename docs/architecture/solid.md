# SOLID

## Definition

SOLID is a set of five object-oriented design principles that help make code easier to maintain and extend.

## The five principles

- Single Responsibility Principle
- Open/Closed Principle
- Liskov Substitution Principle
- Interface Segregation Principle
- Dependency Inversion Principle

## Interview tip

Do not only decode the letters. Give a short practical example for at least one or two principles.

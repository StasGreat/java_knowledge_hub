# Exceptions

## Definition

An exception is an object that represents an error or unexpected condition during program execution.

## Main categories

- checked exceptions
- unchecked exceptions (`RuntimeException` and its children)
- errors

## Example

```java
try {
    Integer value = Integer.parseInt(input);
} catch (NumberFormatException ex) {
    System.out.println("Invalid number");
}
```

## Practical rule

Use checked exceptions for recoverable situations that callers are expected to handle. Use unchecked exceptions for programming errors or invalid state.

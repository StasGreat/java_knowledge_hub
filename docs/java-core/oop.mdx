# OOP Basics

## Definition

Object-oriented programming is a style of programming where data and behavior are modeled together in objects.

## Four pillars

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Simple explanation

OOP helps organize code around real domain objects. Instead of writing unrelated functions and variables, we create classes that represent entities and give them fields and methods.

## Example

```java
class Animal {
    void speak() {
        System.out.println("sound");
    }
}

class Dog extends Animal {
    @Override
    void speak() {
        System.out.println("bark");
    }
}
```

## Common confusion

- **Abstraction** is about hiding unnecessary complexity.
- **Encapsulation** is about hiding internal state and controlling access to it.

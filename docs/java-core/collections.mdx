# Collections

## Quick map

- `List` - ordered, allows duplicates
- `Set` - unique elements
- `Map` - key/value pairs
- `Queue` - processing order

## Example

```java
List<String> names = new ArrayList<>();
names.add("Stas");
names.add("Anna");

Set<String> unique = new HashSet<>(names);
```

## How to choose

Choose a collection based on ordering, duplicates, lookup speed, sorting, and whether you need keys.

# Collections

## Quick map

- `List` - ordered, allows duplicates
- `Set` - unique elements
- `Map` - key/value pairs
- `Queue` - processing order

## Example

```java
List<String> names = new ArrayList<>();
names.add("Stas");
names.add("Anna");
```

## How to answer briefly in an interview

A collection is a framework of interfaces and classes for storing and working with groups of objects.
I choose the implementation depending on ordering, duplicates, lookup speed, and sorting needs.

## Practice task

Explain when you would choose `HashMap` instead of `TreeMap`.

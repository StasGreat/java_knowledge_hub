# Java Knowledge Hub

This project is organized around a simple loop:

1. Learn the theory
2. Check yourself with a topic quiz
3. Do small practice tasks
4. Track completion on the Progress page

## Recommended topic structure

- definition
- short explanation in simple words
- example code
- common mistakes
- topic quiz
- practice tasks

## Suggested learning order

1. Java Core
2. Backend fundamentals
3. Spring
4. Database
5. Architecture
6. Final quiz

## Progress rules

A topic is considered completed when at least one of these is true:

- you click **Mark completed** on the topic page
- your best quiz score for the topic is **90% or more**

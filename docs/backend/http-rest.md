# HTTP and REST

## HTTP

HTTP is an application-layer protocol for communication between clients and servers.

## REST

REST is an architectural style for designing network APIs around resources.

## Common methods

- `GET` - read data
- `POST` - create data
- `PUT` - replace data
- `PATCH` - partial update
- `DELETE` - remove data
